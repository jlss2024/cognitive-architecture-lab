# 🧠 Cognitive Architecture Lab

Welcome to the Cognitive Architecture Lab.

This repository is dedicated to researching how human cognition and artificial intelligence acquire, organize, represent and apply knowledge.

## Mission

Develop a systematic framework that integrates:

- Cognitive Science
- Neuroscience
- Learning Science
- Artificial Intelligence
- Knowledge Representation
- Mental Models

- 05 - Knowledge Representation
- 06 - Experiments
- 07 - Research Notes
